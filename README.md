## 描述

本项目提供了常见的字典查询服务【目前35种】【后期不定时更新】

支持本地部署、远程直接访问

```
民族
专业代码
行业代码
职业工种
地区
```





## 部署(use)

项目支持本地部署、远程部署

### 本地服务

克隆项目

```
git clone https://github.com/rosercoder/dict-search
```

导入数据库文件

```
mysql> source res/dict.sql
```



修改数据库配置

运行项目

```bash
mvn spring-boot:run
```

访问项目

http://127.0.0.1:5444



### 远程服务

项目在

提供接口，可直接访问，如何访问[参数等等]，查询

http://www.wangshuo.cool:5444
